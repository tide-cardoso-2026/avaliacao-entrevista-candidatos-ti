## JUNIOR

[UX][JUNIOR][Diferença entre UX e UI][UX é experiência e jornada; UI é interface visual e interação tangível]
[UX][JUNIOR][O que é persona?][Representação de grupo de usuários com necessidades e contexto; não substitui pesquisa real]
[UI][JUNIOR][Hierarquia visual][Tamanho contraste posição guiam atenção; consistência de componentes]

## PLENO

[UX][PLENO][Jornada do usuário][Etapas emoções pontos de contato; identifica fricção além de uma tela]
